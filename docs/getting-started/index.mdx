# Getting Started

---

Install Cijul as a development dependency.

```sh
npm install --save-dev @nearform/cijul
```

Import the rules you'd like to use.

```js
import {
  background,
  border,
  breakpoints,
  colors,
  spacing,
  text
} from '@nearform/cijul'
```

Create visual primitives using your favorite CSS-in-JS solution and use them in your React components.

## Styled Components

<!-- <StyledComponentsExample>Hello</StyledComponentsExample> -->

```jsx
import React from 'react'
import styled from 'styled-components/macro'
import { color, text } from '@nearform/cijul'

const StyledHeader = styled.h1`
  ${color.red}
  ${text.size.large}
`

export const Header = ({ children }) => <StyledHeader>{children}</StyledHeader>
```

[Further Styled Components integration options](/styled-components)

## Emotion

<!-- <EmotionExample>Hello</EmotionExample> -->

```jsx
/** @jsx jsx */
import { jsx, css } from '@emotion/core'
import { color, text } from '@nearform/cijul'

export const Header = ({ children }) => (
  <h1
    css={css`
      ${color.red}
      ${text.size.large}
    `}
  >
    {children}
  </h1>
)
```
