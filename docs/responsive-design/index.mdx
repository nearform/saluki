# Responsive Design

---

## Example

<Breakpoint>Hello</Breakpoint>

```js
const Text = styled.h1`
  ${color.red}
  ${breakpoint.medium(color.blue)}
  ${breakpoint.large(color.green)}
`
```

## Rules

```
breakpoint.notSmall()
breakpoint.medium()
breakpoint.large()
```

## Configuration

Configurable via;

```
breakpoint
```

## Defaults

```js
{
  notSmall: '576px',
  medium: { min: '768px', max: '991px' },
  large: '992px'
}
```
