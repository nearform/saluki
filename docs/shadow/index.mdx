import { Shadow } from './'

# Shadow

---

## Example

<Shadow />

```js
const Box = styled.h1`
  ${size.height.medium}
  ${background.black}
  ${shadow.large}
`
```

## Rules

```js
shadow.small
shadow.medium
shadow.large
```

## Configuration

Configurable via;

```
shadow
```

## Defaults

```js
{
  small: '0 1px 3px rgba(0,0,0,0.12), 0 1px 2px rgba(0,0,0,0.24)',
  medium: '0 3px 6px rgba(0,0,0,0.16), 0 3px 6px rgba(0,0,0,0.23)',
  large: '0 14px 28px rgba(0,0,0,0.25), 0 10px 10px rgba(0,0,0,0.22)'
}
```
