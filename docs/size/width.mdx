# Width

---

## Example

<Width>Hello</Width>

```js
const Width = styled.div`
  ${size.width.half}
  ${background.red}
`
```

## Rules

```js
size.width.sixth
size.width.fifth
size.width.quarter
size.width.third
size.width.half
size.width.full
size.width.view
```

## Configuration

Configurable via;

```
size.width
```

## Defaults

```js
{
  size: {
    width: {
      sixth: '16.6666666667%',
      fifth: '20%',
      quarter: '25%',
      third: '33.3333333333%',
      half: '50%',
      full: '100%',
      view: '100vw'
    }
  }
}
```
