# Width

---

## Example

<Width>Hello</Width>

```js
const Box = styled.div`
  ${size.width.half}
  ${background.color.red}
`
```

## Rules

```js
size.width.fifth
size.width.twoFifths
size.width.threeFifths
size.width.fourFifths
size.width.quarter
size.width.threeQuarters
size.width.third
size.width.twoThirds
size.width.half
size.width.full
size.width.view
size.width.auto
```

## Configuration

Configurable via;

```
size.width
```
