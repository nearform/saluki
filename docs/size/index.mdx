# Size

---

## Rules

```js
size.width.sixth
size.width.fifth
size.width.quarter
size.width.third
size.width.half
size.width.full
size.width.view

size.height.small
size.height.medium
size.height.large
```

## Configuration

Configurable via;

```
size
```
