# Height

---

## Example

<Height />

```js
const Box = styled.div`
  ${size.height.medium}
  ${background.color.black}
`
```

## Rules

```js
size.height.small
size.height.medium
size.height.large
size.height.quarter
size.height.half
size.height.threeQuarters
size.height.full
size.height.view
size.height.auto
```

## Configuration

Configurable via;

```
size.height
```

## Defaults

```js
{
  small: '2rem',
  medium: '4rem',
  large: '8rem',
  quarter: '25%',
  half: '50%',
  threeQuarters: '75%',
  full: '100%',
  view: '100vw',
  auto: 'auto'
}
```
