## Example

<Height />

```js
const Box = styled.div`
```

## Rules

```js
size.height.small
size.height.medium
size.height.large
```

## Configuration

Configurable via;

```
size.height
```

## Defaults

```js
{
  small: '1rem',
  medium: '3rem',
  large: '6rem'
}
```
