# Overrides

---

You can override a default by matching the rule's structure and providing your own value.

```js
import { init } from '@nearform/saluki'

const salukiConfig = {
  color: {
    red: '#bd0000'
  },
  text: {
    size: {
      medium: '1.5rem'
    },
    serif: 'georgia, times, serif'
  }
}

init(salukiConfig)
```
