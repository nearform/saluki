# Additions

---

You can add new rules by adding to existing parent rules.

Default rules will stay intact unless overriden.

```js
import { init } from 'saluki'

const salukiConfig = {
  color: {
    lightestRed: '#fff5f5'
  },
  text: {
    weight: {
      thick: 900
    }
  },
  breakpoint: {
    extraLarge: '1600px'
  }
}

init(salukiConfig)
```
