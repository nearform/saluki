import { Opacity } from './'

# Opacity

---

## Example

<Opacity>Hello</Opacity>

```js
const Text = styled.h1`
  ${opacity.half}
`
```

## Rules

```js
opacity.full
opacity.half
opacity.none
opacity.inherit
```

## Configuration

Configurable via;

```
opacity
```

## Defaults

```js
{
  full: 1,
  half: 0.5,
  none: 0,
  inherit: 'inherit'
}
```
