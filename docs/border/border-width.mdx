import { Width } from './'

# Border Width

---

## Example

<Width>Hello</Width>

```js
const Text = styled.h1`
  ${border.style.solid}
  ${border.width.thick}
`
```

## Rules

```js
border.width.none
border.width.thin
border.width.medium
border.width.thick
border.width.extraThick
```

## Configuration

Configurable via;

```
border.width
```

## Defaults

```js
{
  none: 'none',
  thin: '1px',
  medium: '2px',
  thick: '4px',
  extraThick: '8px'
}
```
