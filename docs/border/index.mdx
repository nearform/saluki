# Border

---

## Rules

```js
border.color.black
border.color.blue
border.color.gray
border.color.green
border.color.orange
border.color.red
border.color.purple
border.color.yellow
border.color.white

border.width.none
border.width.thin
border.width.medium
border.width.thick
border.width.extraThick

border.style.solid
border.style.dashed
border.style.dotted
border.style.none

border.radius.small
border.radius.medium
border.radius.large
```

## Configuration

Configurable via;

```
border
```

## Defaults

```js
{
  width: {
    none: 'none',
    thin: '1px',
    medium: '2px',
    thick: '4px',
    extraThick: '8px'
  },
  radius: {
    small: '2px',
    medium: '4px',
    large: '6px'
  }
}
```
