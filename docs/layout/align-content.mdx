# Align Content

---

## Example

<AlignContent>
  <AlignContentBox light />
  <AlignContentBox />
</AlignContent>

```jsx
const Container = styled.div`
  ${size.height.large}
  ${layout.wrap.wrap}
  ${layout.display.flex}
  ${layout.alignContent.between}
  ${background.color.yellow}
`

const Box = styled.div`
  ${size.height.small}
  ${size.width.full}
  ${({ light }) => (light ? background.color.red : background.color.black)}
`
```

## Rules

```js
layout.alignContent.start
layout.alignContent.end
layout.alignContent.center
layout.alignContent.between
layout.alignContent.around
```
