# Align Self

---

## Example

<AlignSelf>
  <AlignSelfBox stretch />
  <AlignSelfBox />
  <AlignSelfBox stretch />
  <AlignSelfBox />
</AlignSelf>

```jsx
const Container = styled.div`
  ${size.height.large}
  ${layout.display.flex}
  ${background.color.yellow}
`

const Box = styled.div`
  ${({ stretch }) => (stretch ? layout.alignSelf.stretch : size.height.medium)}
  ${size.width.quarter}
  ${({ stretch }) => (stretch ? background.color.red : background.color.black)}
`
```

## Rules

```js
layout.alignSelf.start
layout.alignSelf.end
layout.alignSelf.stretch
layout.alignSelf.center
layout.alignSelf.auto
```
