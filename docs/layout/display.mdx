# Display

---

## Example

<Container>
  <Box light />
  <Box />
</Container>

```jsx
const Container = styled.div`
  ${layout.display.flex}
`

const Box = styled.div`
  ${size.height.medium}
  ${size.width.half}
  ${({ light }) => (light ? background.color.red : background.color.black)}
`
```

## Rules

```js
layout.display.block
layout.display.inlineBlock
layout.display.flex
layout.display.inlineFlex
```
