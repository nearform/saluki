import { Weight } from './'

# Text Weight

---

## Example

<Weight>Hello</Weight>

```js
const Text = styled.h1`
  ${text.weight.light}
`
```

## Rules

```js
text.weight.thin
text.weight.normal
text.weight.bold
```

## Configuration

Configurable via;

```
text.weight
```

## Defaults

```js
{
  thin: '200',
  normal: 'normal',
  bold: 'bold'
}
```
