import { LetterSpacing } from './'

# Letter Spacing

---

## Example

<LetterSpacing>Hello</LetterSpacing>

```jsx
const Text = styled.h1`
  ${text.letterSpacing.large}
`
```

## Rules

```js
text.letterSpacing.small
text.letterSpacing.none
text.letterSpacing.medium
text.letterSpacing.large
```

## Configuration

Configurable via;

```
text.letterSpacing
```

## Defaults

```js
{
  small: '-.05em',
  none: '0',
  medium: '.025em',
  large: '.05em'
}
```
