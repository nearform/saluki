import { Align } from './'

# Text Align

---

## Example

<Align>Hello</Align>

```js
const Text = styled.h1`
  ${text.align.center}
`
```

## Rules

```js
text.align.center
text.align.left
text.align.right
text.align.justify
```

## Configuration

Configurable via;

```
text.align
```

## Defaults

```js
{
  text: {
    align: {
      center: 'center',
      left: 'left',
      right: 'right',
      justify: 'justify'
    }
  }
}
```
