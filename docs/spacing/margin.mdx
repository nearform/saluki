```js
spacing.margin.small
spacing.margin.medium

spacing.margin.horizontal.large

spacing.margin.top.small

spacing.margin.top.large

spacing.margin.right.small
spacing.margin.right.medium
spacing.margin.right.large

spacing.margin.bottom.small

spacing.margin.bottom.large

spacing.margin.left.small

spacing.margin.left.large
```

## Configuration

Configurable via;

```
spacing.margin
```

## Defaults

```js
{
  small: '1rem',
  medium: '2rem',
  large: '3rem'
}
```
