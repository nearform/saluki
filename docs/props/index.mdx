# Props

Styling based on props can be achieved in a variety of different ways depending on how you're using your CSS-in-JS solution.

## Styled

<!--
<StyledComponentsStyled>Default</StyledComponentsStyled>
<StyledComponentsStyled invert>Inverted</StyledComponentsStyled>

<EmotionStyled>Default</EmotionStyled>
<EmotionStyled invert>Inverted</EmotionStyled>
-->

```jsx
const Text = styled.h1`
  ${props => props.invert && color.white}
  ${props => (props.invert ? background.color.black : background.color.white)}
`
```

## CSS Prop

<!--
<StyledComponentsCSSProps>Default</StyledComponentsCSSProps>
<StyledComponentsCSSProps invert>Inverted</StyledComponentsCSSProps>

<EmotionCSSProps>Default</EmotionCSSProps>
<EmotionCSSProps invert>Inverted</EmotionCSSProps>
-->

```jsx
const Text = ({ children, invert }) => {
  return (
    <h1
      css={css`
        ${invert && color.white}
        ${invert ? background.color.black : background.color.white}
      `}
    >
      {children}
    </h1>
  )
}
```

## Array

<!--
<StyledComponentsArray>Default</StyledComponentsArray>
<StyledComponentsArray invert>Inverted</StyledComponentsArray>

<EmotionArray>Default</EmotionArray>
<EmotionArray invert>Inverted</EmotionArray>
-->

```jsx
const Text = ({ children, invert }) => (
  <h1
    css={[
      invert && color.white,
      invert ? background.color.black : background.color.white
    ]}
  >
    {children}
  </h1>
)
```

## Object Literal

<!--
<StyledComponentsObjectLiteral>Default</StyledComponentsObjectLiteral>
<StyledComponentsObjectLiteral invert>Inverted</StyledComponentsObjectLiteral>

<EmotionObjectLiteral>Default</EmotionObjectLiteral>
<EmotionObjectLiteral invert>Inverted</EmotionObjectLiteral>
-->

```jsx
const Text = styled.h1(props => ({
  ...(props.invert && color.white),
  ...(props.invert ? background.color.black : background.color.white)
}))
```
