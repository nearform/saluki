# Styled Components

---

There are various ways Saluki can be used alongside Styled Components.

## Tagged Template Literal

<!-- <TaggedTemplateLiteral>Hello</TaggedTemplateLiteral> -->

```jsx
import React from 'react'
import styled from 'styled-components/macro'
import { color, text } from '@nearform/saluki'

const StyledHeader = styled.h1`
  ${color.red}
  ${text.size.large}
`

export const Header = ({ children }) => <StyledHeader>{children}</StyledHeader>
```

## CSS Prop

<!-- <CSSProp>Hello</CSSProp> -->

```jsx
import React from 'react'
import { color, text } from '@nearform/saluki'

export const Header = ({ children }) => (
  <h1
    css={`
      ${color.red}
      ${text.size.large}
    `}
  >
    {children}
  </h1>
)
```

## CSS Prop Array

<!-- <CSSPropArray>Hello</CSSPropArray> -->

```jsx
import React from 'react'
import { color, text } from '@nearform/saluki'

export const Header = ({ children }) => (
  <h1 css={[color.red, text.size.large]}>{children}</h1>
)
```

## CSS Prop Object Literal

<!-- <CSSPropObjectLiteral>Hello</CSSPropObjectLiteral> -->

```jsx
import React from 'react'
import { color, text } from '@nearform/saluki'

export const Header = ({ children }) => (
  <h1
    css={{
      ...color.red,
      ...text.size.large
    }}
  >
    {children}
  </h1>
)
```

## Object Literal

<!-- <ObjectLiteral>Hello</ObjectLiteral> -->

```jsx
import React from 'react'
import styled from 'styled-components/macro'
import { color, text } from '@nearform/saluki'

const StyledHeader = styled.h1({
  ...color.red,
  ...text.size.large
})

export const Header = ({ children }) => <StyledHeader>{children}</StyledHeader>
```
